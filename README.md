# Koa2 TS Template

> 基于 TypeScript 的 Koa2 后端脚手架，集成 ESlint、Prettier、TypeORM、PM2等库在内

## Quick Strat

```bash
yarn
```

> 本地需要配置 `.env`文件

```bash
yarn start
```

## Build

```bash
yarn build
```

## Production

### 启动服务

```bash
yarn pro
```

### 查看日志

```bash
pm2 logs -f
```

### 关停服务

```bash
yarn stop
```
